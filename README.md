# Multiplayer Heardle

A real-time multiplayer music guessing game inspired by Heardle and Skribbl.io, built with React, Firebase Functions, and Socket.IO.

## Monorepo Structure

- `frontend/` – React app source code
- `backend/` – Firebase Functions backend
- `shared/` – Shared types and utilities
- `public/` – Static assets for Firebase Hosting
